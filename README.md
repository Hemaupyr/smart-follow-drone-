# Smart Follow Drone – Budget Build

This is a low-cost smart drone project built under ₹10,000 budget. 
The drone supports basic flight, object following using ultrasonic sensor, 
photo capture using ESP32-CAM, and Bluetooth-based control.

## Features
- Quadcopter drone build
- Object follow mode (HC-SR04 sensor)
- Photo capture and WiFi streaming (ESP32-CAM)
- Bluetooth joystick control
- Budget-friendly components
- DIY assembly

## Hardware Used
- 450mm Quadcopter Frame
- 2212 1000KV Brushless Motors
- 30A ESC
- CC3D Flight Controller
- ESP32-CAM
- HC-SR04 Ultrasonic Sensor
- Arduino
- HC-05 Bluetooth Module
- 3S LiPo Battery

## Software Used
- Arduino IDE
- OpenPilot / LibrePilot (CC3D setup)
- ESP32 Camera Web Server
- Bluetooth Controller App

## Applications
- Student drone project
- Embedded systems learning
- Robotics portfolio
- Final year demo
